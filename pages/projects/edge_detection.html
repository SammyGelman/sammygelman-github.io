Edge Detection

    A couple of months ago I was hired by a company trying to automate the mushroom picking process. There were many hurdles to overcome, the one I focused on was the identification and catagorization of mushrooms using computer vision and clever data architecture.

    While in the interview process I made this repo where I implemented an edge detection algorithm called the Hough Circle Transform. Included in the repository is a means of generating testing data and a script for testing the efficiency of the algorithm. The below PDF is a write up of the project as well as a summary of the results. Feel welcome to clone the repository and run the code yourself!
